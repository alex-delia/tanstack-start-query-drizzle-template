A Tanstack Start template with Tanstack Query and Drizzle preconfigured. This template is optimized to use Railway's private networking for faster speeds and lower costs. To take full advantage of private networking DB fetching must use a Server Function.

## Development

From your terminal:

```sh
npm install
npm run dev
```
